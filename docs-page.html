Introduction Last updated: 2019-06-01

MiniJs:- Your Gateway to Lightweight and Effortless JavaScript Magic! Embark on a seamless journey into the world of web development with MiniJs, a feather-light and user-friendly JavaScript library. Designed with simplicity and efficiency in mind, MiniJs leverages the power of the Observable-slim library to effortlessly detect changes within objects. What sets MiniJs apart is its ingenious approach - navigating through object paths, it dynamically identifies associated elements using custom attributes, enabling you to execute operations with unparalleled ease.

Say goodbye to complexity and hello to a coding experience that is both intuitive and swift. MiniJs is your trusted companion for creating dynamic, responsive, and interactive web applications. Join us as we unravel the simplicity and potential encapsulated within the MiniJs Library, where lightweight meets high performance.

Showing and Hiding Elements

One of the key features of MiniJS is its ability to dynamically show or hide elements based on specified conditions. This can be particularly useful for creating responsive and interactive user interfaces.

Usage

To show or hide elements using MiniJS, you can modified your properties to true or false.

						
//html

<div minijs-if="cond">if cond is true</div>
<div minijs-if="!cond">if cond is false</div>

const instance = MiniJs.create({
	cond: true, 
});
instance.cond = false;

We can write any expression to minijs-if. it will evalute and show and hide based on result

String Interpolation

MiniJs simplifies dynamic content population through string interpolation. This feature allows you to bind and update text content.

Usage

To use string interpolation in MiniJs, you can leverage the minijs-text attribute in your HTML. This attribute associates an element with a specific key, and its content will be dynamically updated when the associated key changes.

						
//html
<div>
	<p minijs-text="greeting"></p>				//Hello
	<p minijs-text="{{greeting}} John!"></p>		//Hello John
</div>

//js
const instance = MiniJs.create({
    greeting: 'Hello', 
});

Updating Dynamic Attributes

MiniJs simplifies the process of dynamically updating HTML attributes with the minijs-attr directive. This feature allows you to bind and update attributes dynamically based on specified values, providing a flexible way to manipulate elements in your web application.

Usage

To use the dynamic attribute updating feature in MiniJs, you can leverage the minijs-attr attribute in your HTML. This attribute allows you to specify attribute-value pairs that will be dynamically updated based on changes in your data object.

Example
						

<div>
  <p minijs-attr="'data-id' = dataId">Element with dynamic data-id attribute</p>
  <img src="placeholder.jpg" alt="Image" minijs-attr="'src' = imageUrl; 'alt' = imageAlt">
</div>


const instance = MiniJs.create({
    imageUrl: "new_image.jpg",
    imageAlt: "New Image Alt Text",
    dataId: "1234"
})

Updating Dynamic Classes

MiniJs simplifies dynamic class management by providing the minijs-class directive. This feature allows you to conditionally apply or remove CSS classes based on specified conditions, enabling dynamic styling in your web application.

Usage

To use dynamic class updating in MiniJs, leverage the minijs-class attribute in your HTML. This attribute allows you to specify class-condition pairs that will be dynamically applied or removed based on changes in your data object.

Example
						

<div>
  <div minijs-class="'active' = data.isActive">Dynamic Class Based on Condition</div>
  <button minijs-class="'btn-primary' = isPrimary, 'btn-secondary' = !isPrimary">Toggle Primary/Secondary</button>
</div>


const instance = MiniJs.create({
    isPrimary: true
})

minijs-class="'className' = expression"

you can use any type of expression, it will evalute and based on that result, it will toggle your class

Dynamically Disabling Elements

MiniJs provides a convenient way to dynamically disable HTML elements using the minijs-disabled directive. This feature enables you to conditionally enable or disable user interactions on elements based on specified expressions.

Usage

To leverage the dynamic disabling feature in MiniJs, use the minijd-disabled attribute in your HTML. This attribute allows you to specify an expression that evaluates to a boolean value. If the expression evaluates to true, the element will be disabled; otherwise, it will be enabled.

Example
						

<button minijs-disabled="isDisabled">Click me</button>
<input type="text" minijs-disabled="inputDisabled">


const instance = MiniJs.create({
    isDisabled: true
})

Iterating Over Elements with MiniJs

MiniJs simplifies the process of dynamically creating HTML elements through iteration using the minijs-for directive. This feature enables you to loop through a list and generate elements based on the specified template.

Usage

To utilize the iteration feature in MiniJs, use the minijs-for attribute in your HTML. This attribute allows you to specify a list, and for each item in the list, MiniJs will dynamically generate elements based on the provided template.

Example
						

<div minijs-for="list" minijs-let="item">
	<p minijs-text="item"></p>
</div>

const instance = MiniJs.create({
    list: ['Apple', 'Banana', 'Orange', 'Grapes']
})

In this example, the minijs-for attribute is used to specify the list to iterate over, and the minijs-let attribute is used to define the template for each item.

Two-Way Data Binding with MiniJs

MiniJs simplifies the implementation of two-way data binding for input fields using the minijs-input directive. This feature allows you to seamlessly bind the value of an input field to a variable in your data object, ensuring automatic updates in both directions.

Usage

To enable two-way data binding with MiniJs, use the minijs-input attribute in your HTML. This attribute associates an input field with a variable in your data object, creating a dynamic link that keeps both the input field and the data object in sync.

Example
						

<input input type="text" minijs-input="userName">
<p minijs-text="userName"></p>

const instance = MiniJs.create({
    userName: "John Doe"
})

In this example, the minijs-input attribute is used to bind the value of the input field to the userName variable in the data object.

Parent Selector

By Default it searches inside your whole application to perform operation when ever object data changed.
but you can changed it using parentSelector options to boost your application performance


const instance = MiniJs.create({
	 name: "john Doe"
   }, 
   {parentSelector = '#main-container'}
  )

now your name field only works inside main-container elements
